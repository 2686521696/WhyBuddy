# WhyBuddy Skill Effect Preview

## 用户看到的效果

- 输入一句话目标后，系统先展示归一化摘要与证据来源
- 若缺信息，系统会明确告诉用户哪些是阻塞项、哪些是可稍后补充项
- 系统展示多条路线并解释风险与收益
- 对复杂任务，系统会暴露多角色协作后的综合结论、信心分与分歧意见
- 规格树生成后，用户可以同时看到 requirements、design、tasks 三份文档
- 若上游变更导致产物失效，界面会看到失效原因、来源阶段与自动重算结果
- 若评审不通过，界面会说明系统退回到了哪一层重规划

## 交互反馈

- 所有 gate 都能看到通过与拦下后的去向
- 用户不满意预览时，可以直接触发再推演，而不是重新从头输入
- 超预算或不收敛时，系统提供转人工出口，而不是无限回炉

## 宿主侧呈现建议

- 主视图展示当前阶段和节点状态
- 侧栏展示证据、失效索引、来源追踪
- 回放页按 `sessionId` 查看每轮决策与文档快照
