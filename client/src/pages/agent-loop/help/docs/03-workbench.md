# 工作台指南

工作台（我的应用）是所有推演产物的画廊——每张卡片是一个会话生成的应用。

## 卡片读法

```
┌──────────────────────────┐
│ ●运行中          [缩略图] │  ← 状态徽标 + 按真实模型渲染的迷你应用
├──────────────────────────┤
│ 社区口腔诊所预约与种植牙…  │  ← 话题（应用名）
│ 诊所预约日历·种植病例看板  │  ← 功能标签：真实页面名
│ ▦9 ▤7 ⑂9 ◉7        6/6 │  ← 实体/页面/流程/角色计数 + 闭环证据
└──────────────────────────┘
```

- **缩略图不是截图**：深色导航里是该应用真实的页面名，数据块是真实的实体名——由持久化的五系统模型现场渲染。未闭环的应用显示推演态和证据进度条，不摆假图。
- **三种状态**：`运行中`（闭环 6/6，可打开操作）、`推演中`（还在收敛）、`待补充`（引擎停下来等你回答问题）。

## 工具行

| 控件 | 作用 |
|---|---|
| 搜索框 | 按话题名过滤 |
| 全部 / 推演中 / 可运行 | 图标筛选卡，同时显示各状态数量 |
| 服务健康点 | **点击展开**三服务详情：Node API / Python 推演引擎 / LLM 通道（网关 + 模型） |
| 最近更新 | 点击切换排序方向（最近 ↔ 最早） |

## 卡片操作

- **点卡片** → 激活该会话并进入推演页，应用直接可操作
- **悬停右上「…」→ 删除应用** → 删除该会话（幂等操作，误删的测试会话可以这样清理）
