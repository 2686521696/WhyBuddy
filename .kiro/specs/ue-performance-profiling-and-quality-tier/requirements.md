# 需求文档：性能分档与画质分级

## 简介

本规格定义本地电脑性能检测与画质自动分级机制。系统需检测 GPU 型号、显存、CPU 等硬件信息，自动选择合适的画质档位，并提供阴影、特效、分辨率、宠物数量等维度的分级策略。

## 需求

### 需求 1：硬件性能检测

**用户故事：** 作为系统，我希望自动检测用户的硬件配置，以便推荐最佳画质设置。

#### 验收标准

1. 系统应检测 GPU 型号、显存大小、驱动版本。
2. 系统应检测 CPU 型号、核心数、主频。
3. 系统应检测可用内存大小。
4. 检测结果应在 UE 启动前完成。

### 需求 2：画质自动分级

**用户故事：** 作为用户，我希望系统自动选择适合我电脑的画质设置，以便获得流畅体验。

#### 验收标准

1. 系统应提供三个画质档位：高（High）、中（Medium）、低（Low）。
2. 系统应根据硬件检测结果自动推荐画质档位。
3. 用户应可手动切换画质档位。
4. 画质切换应在 5 秒内生效，无需重启 UE。

### 需求 3：分级策略

**用户故事：** 作为系统，我希望不同画质档位有明确的参数差异，以便在性能和画质之间取得平衡。

#### 验收标准

1. 高画质：全分辨率、Lumen GI、高精度阴影、全特效、最多 6 个角色。
2. 中画质：75% 分辨率、简化 GI、中精度阴影、基础特效、最多 4 个角色。
3. 低画质：50% 分辨率、无 GI、简单阴影、最少特效、最多 2 个角色。
4. 每个档位应保证最低 30fps。
