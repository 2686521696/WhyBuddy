# 任务清单：UE 办公室场景搭建

## 任务

- [ ] 1. 建立场景项目结构与资产规范
  - [ ] 1.1 创建 UE5 项目并配置 Pixel Streaming 插件
  - [ ] 1.2 建立 Content/CubePets/ 目录结构与命名规范文档
  - [ ] 1.3 导入或制作基础家具模型（桌、椅、架、板）

- [ ] 2. 搭建办公室主体空间
  - [ ] 2.1 根据 Three.js OfficeRoom 尺寸建立房间外壳（墙、地板、天花板）
  - [ ] 2.2 布置工位区（4 组办公桌椅 + 显示器）
  - [ ] 2.3 布置会议区（会议桌 + 椅子 + 白板）
  - [ ] 2.4 布置休息区（沙发 + 茶几 + 饮水机）

- [ ] 3. 配置灯光系统
  - [ ] 3.1 设置主光源（Directional Light 模拟日光）
  - [ ] 3.2 添加局部补光（Rect Light / Point Light 模拟台灯和顶灯）
  - [ ] 3.3 创建灯光预设蓝图（日间 / 夜间 / 会议 / 演示）
  - [ ] 3.4 配置 Lumen GI 参数

- [ ] 4. 制作材质系统
  - [ ] 4.1 创建 Master Material 模板（支持 BaseColor / Normal / ORM）
  - [ ] 4.2 制作木材、织物、金属、塑料等材质实例
  - [ ] 4.3 为所有家具分配材质实例

- [ ] 5. 实现 LOD 与性能优化
  - [ ] 5.1 为主要家具模型生成 LOD 层级
  - [ ] 5.2 配置虚拟纹理或 Mipmap 策略
  - [ ] 5.3 在 GTX 1060 上进行性能基准测试

- [ ] 6. 编写资产清单与场景文档
  - [ ] 6.1 生成完整资产清单（模型、材质、贴图列表）
  - [ ] 6.2 编写场景搭建指南与协作规范
  - [ ] 6.3 截图对比 UE5 与 Three.js 场景布局
