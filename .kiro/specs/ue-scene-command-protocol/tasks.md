# 任务清单：场景指令协议

- [ ] 1. 定义指令协议 schema
  - [ ] 1.1 定义 JSON-RPC 2.0 基础消息格式与扩展字段

  - [ ] 1.3 定义错误码枚举与错误响应格式
  - [ ] 1.4 编写协议文档与示例

- [ ] 2. 实现指令路由器与校验层
  - [ ] 2.1 实现 WebSocket 服务端，接收并解析 JSON-RPC 消息

- [ ] 3. 实现指令队列

  - [ ] 3.2 实现指令入队、出队、取消与清空操作
  - [ ] 3.3 实现队列容量限制与溢出拒绝策略
  - [ ] 3.4 编写队列属性测试

- [ ] 4. 实现超时与错误处理
  - [ ] 4.1 为每条指令绑定超时计时器
  - [ ] 4.2 实现超时后自动取消并返回错误响应
  - [ ] 4.3 实现连接断开时的队列清理与通知

- [ ] 6. 编写客户端 SDK
  - [ ] 6.1 封装 TypeScript 客户端，提供类型安全的指令发送方法

  - [ ] 6.3 编写客户端 SDK 使用文档
