## 现有能力落点

当前能力现在同时落在 graph-instance 投影与 runtime 执行面：

- `server/core/web-aigc-controlflow.ts`
  `condition` 节点建模、表达式求值、branchHits 推导、blocked edge 计算，以及运行时表达式校验
- `server/core/workflow-graph-projection.ts`
  把 controlflow 快照挂进统一 graph instance 投影
- `server/core/workflow-runtime-engine.ts`
  把 `condition` 注册为内置运行时适配器，执行成功时推进 conditional edge，执行失败时进入 `EXCEPTION`
- `shared/workflow-graph.ts`
  提供节点运行快照与边状态结构
- `server/tests/workflow-graph-projection.test.ts`
  覆盖正常 condition 分支场景
- `server/tests/workflow-runtime-engine.test.ts`
  覆盖运行时正常条件分支与非法表达式异常场景
